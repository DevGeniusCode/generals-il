# 🎖️ Generals IL - קהילת השחקנים הישראלית ל-Command & Conquer: Generals Zero Hour

ברוכים הבאים לפרויקט האתר של **Generals IL** – מרכז דיגיטלי לקהילת השחקנים הישראלית של המשחק האגדי Command & Conquer: Generals Zero Hour.

האתר נבנה בטכנולוגיית React תוך שימוש ב־Vite, ומציע מדריכים, קישורים, חדשות ומשאבים שימושיים.

---

## 🔗 תצוגה חיה

🖥️ לינק לאתר:  
> [generals-il.vercel.app](https://generals-il.vercel.app)

## 📁 מבנה הפרויקט

```
generals-il/
│
├── public/                  # קבצים סטטיים
├── src/                     # קוד המקור (React)
│   ├── assets/              # תמונות, קבצי markdown, וכו'
│   ├── components/          # קומפוננטות חוזרות
│   ├── pages/               # עמודי תוכן
│   └── App.jsx              # נקודת הכניסה לאפליקציה
│
├── index.html               # קובץ HTML ראשי
├── package.json             # הגדרות npm
├── vite.config.js           # הגדרות Vite
└── README.md                # מדריך זה
```

## 🚀 התקנה והרצה מקומית

1. ודא שמותקן אצלך Node.js (מומלץ 18+)
2. התקנת התלויות:

```bash
npm install
```

3. הרצת סביבת הפיתוח:

```bash
npm run dev
```

4. האתר ייפתח אוטומטית בכתובת:
   [http://localhost:5173](http://localhost:5173)
